# Bug Report: Broken Product Images for Problem User

**Bug ID:** BUG-001
**Status:** OPEN
**Severity:** Medium (Functional/UI)
**Priority:** High (Impacts user trust)

### Environment:
- **URL:** https://www.saucedemo.com
- **Browser:** Chrome
- **User:** `problem_user`

### Steps to Reproduce:
1. Navigate to `https://www.saucedemo.com`.
2. Enter `problem_user` in the Username field.
3. Enter `secret_sauce` in the Password field.
4. Click the **Login** button.
5. Observe the images for the products.

### Expected Result:
Each product should display its own unique and correct image.

### Actual Result:
Every product on the page displays the same identical "dog with a ball" placeholder image.
