#  SauceDemo QA Portfolio Project
**Manual Functional & UI Testing Exploration**

##  Project Overview
This project demonstrates my ability to perform end-to-end manual testing on a standard e-commerce platform (**SauceDemo**). I focused on identifying critical business logic flaws, UI inconsistencies, and account-specific defects.

##  Tech Stack & Tools
* **Testing Type:** Manual Functional & UI Testing
* **Documentation:** Markdown, GitHub Issues (Jira-style tracking)
* **Version Control:** Git & GitHub
* **Environment:** VS Code, Chrome DevTools

## Project Structure
* [**Bug Log**](./bug_log.md): Detailed reports of all identified defects including severity, steps to reproduce, and actual vs. expected results.
* [**Test Cases**](./test_cases.md): A collection of manual test cases covering the login and checkout workflows.
* [**Issues Tab**](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME/issues): Real-time defect tracking and status management.
